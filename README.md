# route-url
Minimal, observable-based URL management library supporting browser, server, and in-memory routing.

## Installation

```bash
npm install
```

```bash
route-url/
├── src/
│   ├── RouteUrl.js
│   ├── BrowserRouteUrlLegacy.js
│   ├── BrowserRouteUrlNavigate.js
│   ├── ServerRouteUrl.js
│   ├── InMemoryRouteUrl.js
│   └── createRouteUrl.js
├── package.json
├── README.md
├── .gitignore
└── vitest.config.js
```
